### [ local network interface 설정 ]
```
- 송신, 수신 host 모두 동일하게 설정
- local network name: TEST.BAE(이런 형식으로 입력)
- Service 20111
- Network Specification ;230.2.1.11;
- import, export subject(TEST.>) 
```

### 2. local network interface 설정 + 테스트(local network name을 TEST.BAE 로 동일하게 설정했을때, 메세지 1개 보냈는데, 1개가 출력됨.), import, export subject(TEST.>) 를 송신, 수신쪽 모두 삭제해서 테스트 했을때도 메세지 1개만 출력
